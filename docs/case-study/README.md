# 设备管理系统开发案例

## 案例概述

本案例展示使用 lnngfar 开发一个完整的**公司设备管理系统**的完整生命周期。

### 案例目标

1. **演示 lnngfar 核心功能**：通过真实项目展示 lnngfar 的设计理念
2. **揭示 lnngfar 功能需求**：在开发过程中自然揭示 lnngfar 需要具备的能力
3. **验证产品设计**：通过实际使用验证产品设计文档的可行性

### 项目背景

- **项目名称**：Company Equipment Management System (CEMS)
- **项目类型**：企业级 Web 应用 + 移动端小程序
- **开发团队**：3-5 人小团队
- **开发周期**：2 个月
- **技术栈**：
  - 后端：Spring Boot + MyBatis Plus + MySQL
  - 前端：Vue3 + TypeScript + Element Plus
  - 移动端：uni-app + uni-ui
  - 部署：Docker + Nginx

## 角色说明

| 角色 | 说明 | 使用的工具 |
|------|------|-----------|
| **产品经理** | 编写需求规格说明书 | Markdown + lnngfar spec |
| **架构师** | 设计数据库和 API | lnngfar entity |
| **后端开发** | 开发 Spring Boot API | lnngfar backend |
| **前端开发** | 开发 Vue3 页面 | lnngfar frontend |
| **移动端开发** | 开发 uni-app 小程序 | lnngfar uniapp |
| **AI 助手** | 全程辅助开发 | lnngfar ai |
