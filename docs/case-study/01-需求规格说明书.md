# 01 - 公司设备管理系统需求规格说明书

## 1. 项目概述

### 1.1 项目背景

某中型科技公司现有员工 200 余人，拥有各类办公设备（电脑、打印机、投影仪等）约 500 台。目前设备管理采用 Excel 表格方式，存在以下问题：

- 设备信息分散，难以统一管理
- 借用归还流程不规范，设备去向不明
- 设备维修记录缺失，无法追溯
- 盘点工作量大，效率低下
- 无法实时了解设备使用状态

### 1.2 项目目标

开发一套公司设备管理系统，实现：

1. **设备台账管理**：统一管理公司所有设备信息
2. **借用归还流程**：规范化设备借用归还流程
3. **维修保养记录**：记录设备维修保养历史
4. **实时盘点功能**：快速了解设备实时状态
5. **移动端支持**：支持扫码快速操作
6. **数据统计分析**：提供设备使用分析报表

#### 2.3.1 借用流程

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│ 发起申请 │ → │ 管理员  │ → │ 领取设备 │ → │ 使用设备 │
│         │    │ 审批    │    │         │    │         │
└─────────┘    └─────────┘    └─────────┘    └─────────┘
     ↓
┌─────────┐
│ 审批拒绝 │
└─────────┘
```

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 借用申请 | 选择设备、填写用途、预计归还时间 | P0 |

```
┌─────────┐    ┌─────────┐    ┌─────────┐
│ 发起归还 │ → │ 管理员  │ → │ 设备入库 │
│         │    │ 验收    │    │ 状态更新 │
└─────────┘    └─────────┘    └─────────┘
```

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 归还申请 | 使用人发起归还申请 | P0 |
| 设备验收 | 管理员验收设备状况 | P0 |
| 异常登记 | 记录设备损坏情况 | P1 |
| 归还确认 | 完成归还，状态更新为空闲 | P0 |

### 3.2 主要数据表

#### 用户表 (sys_user)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键 |
| username | varchar(50) | 用户名 |
| password | varchar(100) | 密码（加密） |
| real_name | varchar(50) | 真实姓名 |
| email | varchar(100) | 邮箱 |
| phone | varchar(20) | 手机号 |
| dept_id | bigint | 部门ID |
| status | tinyint | 状态：0禁用 1启用 |
| create_time | datetime | 创建时间 |
| update_time | datetime | 更新时间 |

#### 设备表 (equipment)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键 |
| code | varchar(50) | 设备编号 |
| name | varchar(100) | 设备名称 |
| category_id | bigint | 分类ID |
| model | varchar(100) | 型号 |
| spec | varchar(200) | 规格参数 |
| price | decimal(10,2) | 购置价格 |
| buy_date | date | 购置日期 |
| status | tinyint | 状态 |
| location | varchar(100) | 存放位置 |
| keeper_id | bigint | 保管人 |
| dept_id | bigint | 归属部门 |
| qr_code | varchar(200) | 二维码 |
| remark | text | 备注 |

#### 借用记录表 (borrow_record)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键 |
| equipment_id | bigint | 设备ID |
| user_id | bigint | 借用人 |
| borrow_date | datetime | 借用时间 |
| expect_return | datetime | 预计归还 |
| actual_return | datetime | 实际归还 |
| status | tinyint | 状态 |
| purpose | varchar(500) | 借用用途 |
| approver_id | bigint | 审批人 |
| approve_time | datetime | 审批时间 |
| remark | varchar(500) | 备注 |
